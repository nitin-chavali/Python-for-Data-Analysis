# Python-for-Data-Analysis
Hands-on practice repository covering NumPy, Pandas, Matplotlib, and Seaborn with real-world data analysis, data manipulation, visualization, and exploratory data analysis (EDA) examples.

## Topics Covered

### NumPy
- Arrays
- Reshaping
- Indexing
- Broadcasting
- Statistical Functions

### Pandas
- DataFrames
- GroupBy
- Merge
- Join
- Pivot Tables
- Missing Values

### Matplotlib
- Line Charts
- Bar Charts
- Histograms
- Subplots

### Seaborn
- Heatmaps
- Pairplots
- Distribution Plots
